Copy this entire assembly hub for local use

This download is only for the purpose of using this assembly hub in your institution which may have firewall access restrictions to this data.
To download the GenArk data for this assembly, use this rsync command: 

  rsync -a -P \
    rsync://hgdownload.soe.ucsc.edu/hubs/GCA/018/852/615/GCA_018852615.2/ \
      ./GCA_018852615.2/
which creates the local directory: ./GCA_018852615.2/
or this wget command: 
  wget --timestamping -m -nH -x --cut-dirs=6 -e robots=off -np -k \
    --reject "index.html*" -P "GCA_018852615.2" \
       https://hgdownload.soe.ucsc.edu/hubs/GCA/018/852/615/GCA_018852615.2/
which creates a local directory: ./GCA_018852615.2/

blat service

There is blat service available for this genome assembly. When viewing this assembly in the genome browser, access the blat service via the Tools -> Blat blue navigation bar menu item.

For local command line blat service, access the blat service via the gfClient command line operation.
See also: hgdownload.soe.ucsc.edu/admin/exe/linux.x86_64/ to download command line binaries.

To operate this locally, you will need the GCA_018852615.2.2bit file from: 

  https://hgdownload.soe.ucsc.edu/hubs/GCA/018/852/615/GCA_018852615.2/
Which can be obtained with rsync via: 
  rsync -a -P 
    rsync://hgdownload.soe.ucsc.edu/hubs/GCA/018/852/615/GCA_018852615.2/GCA_018852615.2.2bit ./
With that GCA_018852615.2.2bit file in your working directory where you run this command, for example, a DNA query with your DNA sequence in the file: someDna.fa with result in the file: GCA_018852615.2.someDna.psl 
gfClient -t=dna -q=dna -genome=GCA_018852615.2 -genomeDataDir=GCA/018/852/615/GCA_018852615.2 
    dynablat-01.soe.ucsc.edu 4040 ./ someDna.fa GCA_018852615.2.someDna.psl
For a protein fasta query with your protein sequence in the file: someProtein.faa with result in the file: GCA_018852615.2.someProtein.psl
