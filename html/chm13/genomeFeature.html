Genome Feature

This track represents the genome features in an abstract way. For details, please refer to the original tracks linked below.

Display Conventions and Configuration

The colors used for each of the genome feature represents:

  •     - CEN, Centromere
  •     - SAT, Satellites
  •     - SD, Segmental Duplications
  •     - SAT_SD, Satellites overlapping with Segmental Duplications
  •     - rDNA, rDNA sequences
  •     - rDNA-MODEL, Model rDNA sequences
  •     - TEL, Telomeric sequences

Methods

Input

  • chm13v2.0_censat_v2.0.bed: Centromere, satellites, and rDNAs
  • New satellite: Satellites newly discovered
  • chm13v2.0_RepeatMasker_4.1.2p1.2022Apr14.bed: chm13v2.0_RepeatMasker_4.1.2p1.2022Apr14.bed
  • chm13v2.0_SD.bed: Segmental Duplications
  • chm13v1.1.rdna_model.bed: Model rDNAs (Chr. 13, 15, 21)
  • chm13v2.0_telomere.bed: Telomeres

Simplifying features

  • CEN: alpha satellite higher-order-repeat annotations within 1kb were merged. Only those over 1kb were used in the simplified CEN annotation.
  • rDNA-MODEL: Chr. 14 and Chr. 22 were determined to have no alternative rDNA alleles, concluding to be no more model sequences. Excluding those chromosomes, all annotations within 100bp were merged for rDNA-MODEL annotation.
  • rDNA: Previously annotated rDNA region were re-used from the censat annotation.
  • TEL: Previously annotated telomere regions were re-used from the telomere annotation.
  • SD: Segmental Duplications (SD) within 1kb were merged. Only those over 50kb were used for further SD annotation. Among the SDs, any SD overlapping the rDNAs or telomeres were discarded. From the rest, anything overlapping the SAT was annotated as SAT/SD.
  • SAT: Satellite annotations from the censat, repeat masker, and new satellites within 1kb were merged and simplified for SAT annotation. Only those with no overlap to features above are annotated as SAT.
