Description

These tracks contain epigenetic analysis of CHM13 and HG002 cells using nanopore sequencing aligned to the CHM13 v2.0 genome

Methods

  • CHM13 nanopore CpG methylation: CHM13 ultra-long nanopore reads were aligned to the CHM13 reference with Winnowmap (v2.0) with a k-mer size of 15. CpG methylation was measured with Nanopolish (v0.13.2) with a log-likelihood ratio (LLR) cutoff of <–1.5 (unmethylated) and >1.5 (methylated).
  • HG002 nanopore CpG methylation: HG002 ultra-long ONT nanopore sequencing data was basecalled with Guppy v6.1.2 using "dna_r9.4.1_450bps_modbases_5mc_cg_sup_prom.cfg" configuration to call CpG methylation, processed with samtools (v1.15.1), aligned to the CHM13 reference with Winnowmap (v2.0.3) with a k-mer size of 15. Aggregated methylation percentages at all CpGs were calculated using modbam2bed (v0.6.2) with bases having >0.8 probability called “methylated” and bases having <0.2 probability called “unmethylated.”
  • HG002 nanoNOMe nanopore CpG methylation: nanoNOMe experiments were performed on HG002 cells as described in Gershman et al. (2022). HG002 ultra-long nanopore nanoNOMe reads were aligned to the CHM13 reference with Winnowmap (v2.0.3) with a k-mer size of 15. CpG methylation was measured with Nanopolish (v0.13.2) with a log-likelihood ratio (LLR) cutoff of <–1.5 (unmethylated) and >1.5 (methylated). Note that this track excludes Cs in GCG motifs due to an enrichment of ambiguous calls (see Lee et al. (2020))
  • HG002 nanoNOMe nanopore GpC methylation: nanoNOMe experiments were performed on HG002 cells as described in Gershman et al. (2022). HG002 ultra-long nanopore nanoNOMe reads were aligned to the CHM13 reference with Winnowmap (v2.0.3) with a k-mer size of 15. GpC methylation was measured with Nanopolish (v0.13.2) with a log-likelihood ratio (LLR) cutoff of <–1.0 (unmethylated) and >1.0 (methylated). Note that this track excludes Cs in GCG motifs (see Lee et al. (2020)). These tracks include methylation percentages and the Z-score of methylation percentages.
